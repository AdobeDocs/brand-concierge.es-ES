# Obtenga información sobre los ID de flujo de datos

Descubra cómo el ID de la secuencia de datos conecta los eventos de Web SDK del sitio con Adobe Brand Concierge en Adobe Experience Platform. Aprenderá por qué se requiere el ID, cómo enruta los datos al entorno limitado y a los conjuntos de datos adecuados, y cómo los equipos utilizan flujos de datos independientes para el desarrollo, el ensayo y la producción.

## ¿Para quién es este vídeo?

* Desarrollador
* Product Manager
* Asesor de soluciones

## Contenido de vídeo

* Razones por las que se requiere el ID de la secuencia de datos para Web SDK y Brand Concierge
* Cómo el flujo de datos enruta eventos a zonas protegidas, conjuntos de datos, esquemas e identidades
* Por qué los equipos utilizan diferentes flujos de datos para el desarrollo, la fase y la producción
